[Glance](https://glnc.io) is a mission control dashboard for AI agents — observability and orchestration for autonomous systems.

When you have AI agents running tasks, processing queues, and making decisions, you need visibility into what they're doing. Glance provides that single pane of glass.

**Features:**
- **Work Queue:** Track tasks across instances, prioritize work, monitor completion
- **Recurring Tasks:** Schedule automated jobs with cron expressions, track run history
- **Activity Feed:** Real-time view of agent actions and decisions
- **Instance Management:** Multiple agents, each with their own context and configuration
- **Cognitive Status:** Monitor agent health, memory state, and operational metrics

Built to manage my own AI assistant (Gomez), but designed to scale to multiple agents and use cases.

The goal: agents that can operate autonomously while giving humans the oversight they need.
